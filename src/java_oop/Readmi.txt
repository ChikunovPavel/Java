ООП - это

Класс это - програмный код
Обьект это - выполененная программа(обьект присваеваеться ссылочной переменной)
    Объект обладает
        Характеристиками (поля. атрибуты)
        Поведением       (методы)

Конструктор -  делаеться для инициализации начального состояния, созднания объекта с уже задаными значениями
        имеет имя класса
        не имеет возврашаемого значения
        если нет конструктора будет использован конструктор по умолчанию


Инкапсуляция - это сокрытие реализация от пользователя
